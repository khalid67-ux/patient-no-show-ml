# AI-Based Patient No-Show Prediction using Machine Learning

## Overview
This project predicts whether a patient will miss a medical appointment using Machine Learning.

## Model
- Logistic Regression

## Dataset
Medical Appointment No Shows (Brazil)

## Features
- Age
- Gender
- Scholarship
- Hypertension
- Diabetes
- SMS_received

## Result
Accuracy: ~80%

## Tools
- Python
- Pandas
- Scikit-learn
- Google Colab
