# AI-Based Patient No-Show Prediction using Machine Learning

## Overview
This research-based project explores how Machine Learning can predict patient no-shows using real-world healthcare data.

## Model
- Logistic Regression

## Dataset
Medical Appointment No Shows (Brazil)

## Features
- Age
- Gender
- Scholarship
- Hypertension
- Diabetes
- SMS_received

## Result
Accuracy: ~80%

## Tools
- Python
- Pandas
- Scikit-learn
- Google Colab
